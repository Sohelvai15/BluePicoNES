Build Guide

1. Flash PicoNES Firmware

Download the latest firmware from:

PicoNES Releases

  • Hold BOOTSEL while connecting Pico to PC.
  • Open the RPI-RP2 drive.
  • Copy the UF2 file.
  • The Pico will reboot automatically.

2. Flash BlueRetro Firmware

BlueRetro Online Flasher

  • Connect ESP32 to PC.
  • Open the flasher page.
  • Select COM Port.
  • Install latest firmware.
  • Restart ESP32.

3. PS3 Controller Pairing

  • Find ESP32 Bluetooth MAC Address using BlueRetro Configurator.
  • Download SixaxisPairTool.
  • Connect PS3 controller via USB.
  • Write ESP32 MAC Address to controller.
  • Disconnect USB and press PS button.
  • Controller will connect wirelessly.
