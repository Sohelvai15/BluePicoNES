# BluePicoNES

BluePicoNES is a DIY retro gaming console based on Raspberry Pi Pico, PicoNES and BlueRetro.

## Features

* NES Emulator (PicoNES)
* Bluetooth Controller Support (BlueRetro)
* PS3 Controller Support
* PS4 Controller Support
* Dual Player Support
* HDMI/DVI Video Output
* microSD ROM Storage
* USB-C Power Input
* Custom Clear Case
* Power Switch
* Bluetooth Enable/Disable Switch

## Hardware Used

* Raspberry Pi Pico
* ESP32-WROOM-32
* Adafruit DVI Breakout
* Adafruit MicroSD Breakout
* USB-C PD Power Module
* USB-A Controller Port
* Schottky Diode
* Capacitors
